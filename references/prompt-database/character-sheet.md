## 使用指南

### 工作流顺序
1. **正视图** → 确认正面脸型和服装正面结构
2. **侧视图** → 确认侧面轮廓和发型侧面
3. **背视图** → 确认背面发型和服装背面
4. **正面特写** → 基于三视图设定生成高质量面部

### 参数替换指南
替换以下变量即可生成不同角色：
- `22 岁年轻女性` → 年龄/性别
- `标准古典鹅蛋瓜子脸` → 脸型描述
- `乌黑长发半挽半散、银色玉簪点缀` → 发型
- `1:8 理想化头身比` → 头身比（女主 1:8 / 男主 1:9）
- `粉色古装露肩纱裙...` → 服装描述

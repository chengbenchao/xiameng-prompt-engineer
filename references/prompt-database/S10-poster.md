## L1: 科普百科信息图（模块化百科卡）

```
独家秘法生成一张全景平拍正面拍摄信息图画面。画面中央为一个清晰漂亮的主题主视觉（蝴蝶标本摄影图），周围分布若干局部特征放大细节（翅膀纹理显微图、生命周期图解、分类信息）。多个圆角模块化信息分区，清楚的标题层级与重点标签。浅色干净背景，柔和配色，轻阴影，精致小图标，圆角信息框，整洁排版，信息密度高但不拥挤。整体像真正可以发布、阅读、收藏的科普百科卡。50mm 镜头，F8.0 光圈，均匀布光，1:1 光比。环境色温 5500K，主体色温 5500K，情绪关键词：知识整理、图鉴感、收藏价值、系列化生产，超写实质感，8k 超高清，IMAX 级渲染 --ar 3:4
负面提示：低画质，模糊，变形，卡通风，3D渲染，广告感，杂乱排版
```

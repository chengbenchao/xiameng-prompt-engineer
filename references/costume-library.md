## 📋 服装条目

### C01 - 魏晋风紫白渐变仙裙

**关键词**：魏晋风汉服 / 紫白渐变 / 仙气 / 轻纱

**上衣**：淡紫色交领窄袖短襦，内衬白色轻纱中衣，袖口有银色滚边

**外披**：透明薄纱外披，大袖宽袍设计，纱衣轻薄半透明，表面可见若隐若现的金色刺绣花纹

**下装**：高腰白色至淡紫色渐变长裙，裙摆拖地，多层轻纱叠加，外层薄纱带银色暗纹，内层纱裙呈现丝绸光泽

**面料**：
- 外层：轻薄透纱，半透明纱质，垂坠感强
- 内层：细腻丝绸，光泽柔和
- 裙摆：多层纱叠加，层次丰富

**配饰**：腰间同色系织锦腰带，悬挂紫玉佩与银色流苏

**色彩**：白色与淡紫色渐变，主色调偏紫白，辅以银色点缀

**适用场景**：S3 宗门白天 / S6 宗门山门 / S4 山谷洞穴（仙侠女主、修炼、出山历练）

**完整服装描述段落（可直接复制入主体第7维）：**
```
她身着一袭魏晋风仙气古装，内穿淡紫色交领短襦与白色轻纱中衣，外罩透明薄纱大袖宽袍，纱衣轻薄半透明，表面若隐若现的金色刺绣花纹。高腰白色至淡紫色渐变长裙，裙摆拖地，多层轻纱叠加，外层薄纱带银色暗纹，内层纱裙呈现细腻丝绸光泽。腰间系同色系织锦腰带，悬挂紫玉佩与银色流苏。
```

### C02 - 少女粉齐胸襦裙

**关键词**：古风 / 少女感 / 甜妹 / 粉色系 / 汉服 / 齐胸襦裙

**色彩**：淡樱花粉，带同色系渐变桃花刺绣

**面料**：外层雪纺（半透轻盈），内层亲肤棉

**上衣**：交领对襟，袖口宽8cm，绣渐变桃花纹样

**下装**：齐胸款，裙摆3米摆，雪纺晕染面料，有自然垂坠感，长度到脚踝上方3cm

**配饰**：腰间系粉色缎带蝴蝶结，肩披同色半透纱质披帛，无多余首饰

**版型**：收腰，长度到脚踝上方3cm

**适用场景**：S3 宗门白天 / S6 宗门山门 / S7 宫廷浴池（少女女主、甜妹风格、春日外景）

**完整服装描述段落（可直接复制入主体第7维）：**
```
她身着一套少女粉齐胸襦裙，上襦为交领对襟款，袖口宽8cm，绣有渐变桃花纹样。下身齐胸款长裙，3米大裙摆采用雪纺晕染面料，淡樱花粉带同色系渐变桃花刺绣，自然垂坠至脚踝上方3cm。腰间系粉色缎带蝴蝶结，肩披同色半透纱质披帛，无多余首饰。
```

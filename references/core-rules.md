## 📷 2. 景别与焦段匹配铁律 (Focal Length Rules)
*   **远景/大远景**：20mm - 24mm (广角，拍大场景)
*   **全景**：24mm - 35mm (人物全身 + 环境)
*   **中全景**：35mm (膝盖以上，带环境)
*   **中景/近景**：40mm - 50mm (标准人像)
*   **特写**：85mm (人像黄金焦段，面部)
*   **大特写**：105mm - 135mm (面部极致压缩，眼睛/嘴部)
*   **⚠️ 违规警告**：特写绝不能用 24mm（会畸变）！全景绝不能用 135mm（拍不全）！

## 💡 3. 光影效果速查 (Lighting Guide)
*   **侧逆光 (Rim Light)**：轮廓光分离，最常用。适合表现人物与背景空间关系。
*   **伦勃朗光 (Rembrandt)**：左上方 45°，三角光斑。适合神秘、反派、有故事感的面孔。
*   **顶光 (Top Light)**：从上往下。适合压抑、诡异、神圣（反派常用）。
*   **正面平光 (Flat Light)**：消除阴影。仅用于三视图、设定图，确保看清结构。
*   **侧顺光 (Soft Key)**：柔和自然。适合温婉、清纯、日常。

## 📐 10. 角色设定图集工作流 (Character Sheet Workflow)

**⚠️ 强制要求**：生成角色设定图集时，必须严格按此工作流执行。

### 10.1 核心原则
1. **先出三视图，后出特写**（不可颠倒）
2. **逐个视角单独生成**，绝不在一张图里拼合多个视角
3. 所有视图的参数除「景别、焦段、拍摄方向」外**必须完全一致**，确保角色一致性

### 10.2 统一参数表（锁定区）

| 维度 | 三视图（正/侧/背） | 特写 |
|------|-------------------|------|
| **景别** | 全身 | 特写 |
| **焦段** | 50mm | 85mm |
| **光圈** | F4.0 | F2.8 |
| **光位** | 正面平光 | 正面平光 |
| **光比** | 1:1 均匀 | 1:1 均匀 |
| **色温** | 5500K / 5500K | 5500K / 5500K |
| **背景** | 纯白无缝背景 | 纯白无缝背景 |

### 10.3 三视图参数差异表

| 视图 | 拍摄方向 | 视角描述关键词 | 面部/发型描述重点 |
|------|---------|---------------|-------------------|
| **正视图** | 正面 0° | `正面视角` | 完整五官，发型正面造型 |
| **侧视图** | 侧面 90° | `侧面 90° 视角` | 侧面轮廓（额头饱满/鼻梁挺直/下颌线），发簪侧面造型，发丝垂落至腰际 |
| **背视图** | 背面 180° | `背面 180° 视角` | 后脑勺圆润饱满，发型背面造型，长发垂落，服装背面细节 |

### 10.4 特写规则
- **景别**从"全身"改为"特写"，**焦段** 85mm
- **拍摄方向**根据需求选择：`正面视角` 或 `斜侧 45° 视角`
- 除景别、焦段外，**脸型、发型、服装描述必须与三视图完全一致**（复制粘贴，不可改写）
- 负面提示额外追加：`阴影，复杂背景`

### 10.5 负面提示扩展

**三视图通用负面提示：**
```
负面提示：低画质，模糊，变形，多余手指，文字水印，动漫风，卡通风，3D渲染，阴影，复杂背景，比例失调，头身比错误
```

**特写通用负面提示：**
```
负面提示：低画质，模糊，变形，多余手指，文字水印，动漫风，卡通风，3D渲染，阴影，复杂背景
```

### 10.6 标准工作流顺序

```
步骤 1：正视图（正面 0°，全身，50mm）
步骤 2：侧视图（侧面 90°，全身，50mm）
步骤 3：背视图（背面 180°，全身，50mm）
步骤 4：特写（正面/斜侧，85mm）
```

**⚠️ 严禁跳步**：必须先完成三视图确认角色设定后，再生成特写。

## 📋 12. 小云雀 (XYQ) 图片生成工作流

**⚠️ 适用场景**：用户要求根据提示词生成图片时，通过小云雀 API 提交生成任务，并直接通过飞书聊天窗口发送图片。

### 12.1 提交任务
```bash
python3 ~/.agents/skills/xyq-nest-skill/scripts/submit_run.py --message "提示词内容"
```
返回 `thread_id`, `run_id`, `web_thread_link`。

### 12.2 轮询结果
```bash
python3 ~/.agents/skills/xyq-nest-skill/scripts/get_thread.py --thread-id <thread_id> --run-id <run_id> --after-seq 0
```
解析返回 JSON 中的 `messages`，找到 `image.url`。

### 12.3 下载图片
```bash
curl -L -o <本地路径> "<图片URL>"
```

### 12.4 发送给用户
使用 `message` 工具的 `media` 参数，通过飞书直接发送图片文件。

### 12.5 关键注意事项
- **角色设定图**：小云雀默认使用 `seedance2.0_fast_vision` 模型，会自动调整提示词
- **AI 可能自行发挥**：眼睛颜色、裙长、款式等可能与提示词有出入，需评审后反馈
- **超时处理**：轮询超时（180秒）后直接调用 get_thread 获取结果
